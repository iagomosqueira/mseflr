Directory structure

data:	Intermediate and final RData objects etc.
db:	Results from analysis in the form of sql databases
docs:	Background documents
figs:	figures produced as part of the analysis, either for conditioning or publication
input:	data used to generate RData objects, e.g. WG data files 
pkgs:	FLR & R packages used for analysis
R:	Sscripts and ad-hoc functions
text:	drafts of papers reports etc.
